## 登出
```
GET /logout
```

参数

```
无
```

返回

```
{
    "code": 0,
    "msg": "success",
    "data": ""
}
```
## 图片上传

```
POST /pic/upload
```
参数
```
file: (file 对象)  // 上传的图片
```

返回
```
{
    "code": 0,
    "msg": "success",
    "data": "119.23.75.180/2018/12/06/3ce0dc69-551d-4ec4-b654-86a1fea432c8.jpeg"
}
```
